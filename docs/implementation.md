#### Definition Of Done

Abychom task považovali za splněný, musí být v rozumné míře splněno:

1. Fukcionalita je hotova.
1. Nad kódem je code review.
1. V případě akvizičního epiku review s bivojem.
1. Je zamergovano.
1. Je nasazeno na test prostredi.
