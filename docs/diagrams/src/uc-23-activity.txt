start

:Dispečer vybere odkaz na zpracování fronty konkrétního dopravce.;

-> Systém je připraven pro práci;

repeat

    :Dispečer naskenuje čárový kód;
    -> Systém vyzve ke zvážení;
    :Dispečer zváží balík;

    if (Odpovídá váha balíku?) then (Ano)
        :Systém vyskne průvodní štítek dopravce;
        :Dispečer nalepí štítek dopravce a
        vloží balík do fronty na předání dopravci;
    else (Ne)
        :Systém požádá o vložení do chybové fronty;
        :Dispečer vloží balík do chybové fronty;
    endif

repeat while (Další balík k zpracování?) is (Ano)
->Ne;

stop
